static文件夹结构

总体说明
  • 本文件夹主要包含pub(发布版本),source(开发源文件),html(静态文件)
  • 在nginx中，将本文件夹设为root，并设置rewrite规则。 
    server {
    listen   80;
    server_name dev.test.kx;
    root E:/workspace/RRSB2BUI/static;
    index index.html;
    charset utf-8;
    rewrite ^/pub/jslib/(.*)$ /source/pub/jslib/$1 last;
    rewrite ^/jslib/(.*)$ /source/jslib/$1 last;
    rewrite ^/pub/csslib/(.*)$ /source/pub/csslib/$1 last;
    rewrite ^/pub/images/(.*)$ /pub/pub/images/$1 last;
}
server {
    listen   80;
    server_name pub.test.kx;
    root E:/workspace/RRSB2BUI/static;
    index index.html;
    charset utf-8;
    rewrite ^/pub/jslib/(.*)$ /pub/pub/jslib/$1 last;
    rewrite ^/jslib/(.*)$ /pub/jslib/$1 last;
    rewrite ^/pub/csslib/(.*)$ /pub/pub/csslib/$1 last;
    rewrite ^/pub/images/(.*)$ /pub/pub/images/$1 last;
}
